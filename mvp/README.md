# MVP PWA личного кабинета

Статический прототип личного кабинета для компании по банкротству физических лиц.

## Что есть

- Главный экран с объяснением “что сейчас происходит”.
- Сгруппированное меню: основные разделы и подразделы без перегруза боковой панели.
- Мои задачи, Мое дело, Документы, Мои долги, Календарь, Сообщения, Платежи, Помощь, Профиль.
- Админ-панель для сотрудников.
- PWA manifest и service worker.
- Файлы для публикации на Netlify, Vercel и GitHub Pages.

## Как открыть

Откройте `index.html` в браузере.

Для полноценной проверки PWA лучше запустить локальный сервер:

```bash
python3 -m http.server 8080
```

И открыть:

```text
http://localhost:8080
```
